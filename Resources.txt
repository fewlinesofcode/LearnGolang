## Libraries and components

- [Gorilla toolkit](https://www.gorillatoolkit.org) is a web toolkit for the Go programming language.
- [Sign-In with Apple](https://github.com/Timothylock/go-signin-with-apple) implementation in Go.
- [Telebot](https://github.com/tucnak/telebot) is a Telegram bot framework in Go.
- [Gin-Gonic](https://gin-gonic.com) on of the most popular Go frameworks for web development. Good for mobile APIs building.
